# bluespice-free
BlueSpice free unofficial native installation package

# Description
This repository is an unofficial release of the BlueSpice MediaWiki free edition.
As the official contributor only offers an installation which depends on docker deployments this project was intended to find a way to install BlueSpice free on native way.

# How the packages are created
The packages for the native installation are created by analyzing the officially released versions of BlueSpice free.
On top more informations came from the official BlueSpice helpdesk since version 3, own experimentation and analyzing source code.

# Differentiation between BlueSpice free and BlueSpice pro
This project will *never* aim to offer more then BlueSpice free.
For the pro edition please only contact the official contributor.
